# ResearchAgent

Automate search of news and content on a topic and generate a brief based on the top results. The brief will include:
- Main themes, headlines, and important text from the sources
- Key facts, data points, and statistics
- Summary
- Source URLs for attribution and dates of the content if from news sources

Also includes an LLM as a judge to ensure that the generated brief satisfies quality criterion (number of sources referenced, quality of output, source citation etc.).
